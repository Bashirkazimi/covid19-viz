COVID19 Dashboard

Visualizations for Up-to-date COVID19 Reports Globally

This project aims to visualize reports for countries with the highest number of COVID19 cases around the globe. Additionally, it reports cases for Afghanistan (my home country), and Bangladesh (my wife's home country). The data is loaded in real time from the daily updated report by Johns Hopkins University on this Github repository.

Global Report
Reports for Afghanistan and Bangladesh

At the first glance, it seems that Aghanistan and Bangladesh are doing pretty well, but I leave it upto the readers to decide whether the cases really are as reported.
